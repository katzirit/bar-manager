Welcome to the BARManeger system!

This system was built as a part of the "Information system planning and analyzis",
in the Azrieli college of engineering.
This system was written in node JS, and the following packages were used:

async
bcrypt-nodejs
body-parser
cookie-parser
express
express-session
lodash - A JavaScript utility library delivering consistency, modularity, performance, & extras.
mysql - A pure node.js JavaScript Client implementing the MySql protocol.
passport - Passport is authentication middleware for Node.js. Extremely flexible and modular.
passport-local
sequelize - Sequelize is a promise-based Node.js/io.js ORM for Postgres, MySQL, MariaDB, SQLite and Microsoft SQL Server.
It features solid transaction support, relations, read replication and more.

This system was designed to utilize the work in a bar.
By inserting known data the bar manager or shift manager will have the needed information to maximize the efficiency and profit of their bar.
By inserting the data the system will run a simulation, written in Java Script, that will output the expected profit, drinks consumption and employee efficiency.
This data will allow the manager to make smarter and more calculated decisions regarding the upcoming shift, which will eventually save him time and money.
